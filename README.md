# PathfinderInGridmap

## 📌 Grid Pathfinding with Smooth Movement Animation

This project is a simple demonstration of pathfinding on a grid-based map. It shows how a character can navigate from one cell to another using a pathfinding algorithm, while also applying smooth movement animation between cells.

### 🚀 Features

* 🧭 **Grid-Based Pathfinding**

  * Finds a path from a starting point to a target position on a grid map.
    
* 🎮 **Smooth Character Movement**

  * Instead of instantly jumping from one cell to another, the character moves smoothly across cells.
   
* 🧩 **Simple and Easy-to-Understand Code**

  * Designed for learning and experimentation.

### 🛠️ Purpose

This project is created as a sample implementation to demonstrate:

* Basic pathfinding logic in a grid environment
* How to animate movement between grid cells for a more natural visual effect

### 📂 How It Works

1. The grid map represents walkable and non-walkable cells.
2. A pathfinding algorithm calculates the shortest path.
3. The character follows the path step-by-step.
4. Movement is interpolated to create smooth transitions between cells.
